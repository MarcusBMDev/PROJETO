# 🚗 NeuroCar

**NeuroCar** é o módulo responsável pela gestão e controle de frota de veículos e transportes do ecossistema Neuro.

Utilizado para gerenciar agendamentos de carros corporativos, motoristas, manutenções, rotas ou mesmo serviços de translado, garantindo controle de gastos e disponibilidade da frota.

## ⚙️ Uso
Este módulo opera sob uma estrutura direta (sem um empacotador listado como `package.json` na raiz). Verifique as configurações de ambiente ou arquivos estáticos para subida em servidor web.

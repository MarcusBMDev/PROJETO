# 🖨️ NeuroPrint

**NeuroPrint** é a solução de gerenciamento de cotas e auditoria de impressões.

Este sistema controla a política de uso consciente de impressoras do grupo Neuro. Ele realiza cálculos avançados para converter documentos, checar páginas frente-e-verso (duplex), contagem por folha e descontar das cotas específicas de cada setor ou colaborador, evitando desperdício de papel e recursos.

## 🛠️ Tecnologias Utilizadas
*   **Node.js**: Backend para manipulação das requisições e cálculos de metadados de impressão.

## ⚙️ Como Executar
1. Acesse a pasta `NEUROPRINT`.
2. Instale as dependências: `npm install`
3. Rode o serviço: `npm start`

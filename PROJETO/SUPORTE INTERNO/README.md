# 🛠️ Suporte Interno

**Suporte Interno** é a central de Helpdesk (chamados) para as equipes de **Tecnologia da Informação (TI)** e **Manutenção Predial**.

É por aqui que os colaboradores registram incidentes com computadores, internet, aparelhos, ar condicionado, energia ou qualquer necessidade de reparo infraestrutural. O sistema ajuda os técnicos a organizarem a fila de atendimento (SLA), prioridades e resoluções.

## 🛠️ Tecnologias Utilizadas
*   **Node.js**: Plataforma central de processamento.

## ⚙️ Como Executar
1. Navegue até a pasta `SUPORTE INTERNO`.
2. Baixe as dependências: `npm install`
3. Rode a aplicação: `npm start`

# 🧩 NeuroABA

**NeuroABA** é o microserviço dedicado ao ecossistema de terapias ABA (Applied Behavior Analysis). 

Este módulo permite o registro, acompanhamento de evolução e estruturação de dados clínicos específicos para pacientes em terapia ABA. O sistema garante que a equipe multidisciplinar tenha um registro centralizado e seguro das intervenções, medições de frequência e progressos.

## 🛠️ Tecnologias Utilizadas
*   **Node.js**: Backend para processamento rápido e não-bloqueante.
*   (Outras tecnologias gerenciadas via `package.json`)

## ⚙️ Como Executar
1. Acesse o diretório do NeuroABA.
2. Instale as dependências: `npm install`
3. Execute o servidor: `npm start` ou `npm run dev` para desenvolvimento.

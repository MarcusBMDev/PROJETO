# 📅 NeuroAgenda

**NeuroAgenda** é o microserviço responsável pela centralização de compromissos, reuniões e alinhamentos de equipes do grupo Neuro.

Embora o *NeuroGestão* lide com as marcações de pacientes e grades de especialistas, o **NeuroAgenda** pode atuar na organização corporativa, reservas de salas de reunião, eventos internos e integrações de calendário.

## 🛠️ Tecnologias Utilizadas
*   **Node.js**: Plataforma de backend baseada no `package.json`.

## ⚙️ Como Executar
1. Acesse o diretório do NeuroAgenda.
2. Instale as dependências: `npm install`
3. Inicie a aplicação: `npm start`

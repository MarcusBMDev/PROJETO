# 🛒 NeuroCompras

**NeuroCompras** é o sistema centralizado de requisições, controle de estoque e aprovação de suprimentos do grupo Neuro.

Facilita o fluxo logístico entre os setores (como limpeza, copa, escritório, materiais clínicos) e o setor de Compras, automatizando aprovações, cotações e status de entrega, garantindo que nunca faltem insumos para a clínica.

## 🛠️ Tecnologias Utilizadas
*   **Node.js**: Plataforma principal orientada pelo `package.json`.

## ⚙️ Como Executar
1. Acesse a pasta `NEUROCOMPRAS`.
2. Instale os pacotes: `npm install`
3. Inicie o projeto: `npm start`

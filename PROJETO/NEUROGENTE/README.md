# 👥 NeuroGente

**NeuroGente** é o portal do colaborador e sistema de gestão de pessoas (RH).

O sistema centraliza as informações do quadro de funcionários da clínica, oferecendo um cockpit com dados úteis, lista de aniversariantes, avisos corporativos e a central de **Benefícios** (com acessos diretos para convênios como SESC, academias, faculdades, etc).

## 🛠️ Tecnologias Utilizadas
*   **Node.js**: Aplicação backend.

## ⚙️ Como Executar
1. Entre na pasta `NEUROGENTE`.
2. Execute `npm install` para instalar as bibliotecas.
3. Inicie o projeto: `npm start` ou `node index.js` (conforme `package.json`).

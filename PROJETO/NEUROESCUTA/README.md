# 🎧 NeuroEscuta

**NeuroEscuta** é o microserviço dedicado ao acolhimento e canais de escuta (psicológica, RH ou Ouvidoria).

O sistema permite que colaboradores ou clientes enviem feedbacks, solicitações confidenciais ou agendem momentos de escuta profissional de forma segura, garantindo o registro e triagem para os profissionais capacitados do grupo.

## 🛠️ Tecnologias Utilizadas
*   **Node.js**: Base da aplicação.

## ⚙️ Como Executar
1. Acesse a pasta `NEUROESCUTA`.
2. Instale os módulos: `npm install`
3. Execute: `npm start`

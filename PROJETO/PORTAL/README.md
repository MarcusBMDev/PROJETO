# 🌌 Portal Neuro

**Portal** é a "Porta de Entrada" (Hub) para o ecossistema de microserviços do NeuroSystem.

Trata-se de uma interface visual unificada (Intranet/Dashboard) por onde os colaboradores podem fazer login e acessar todos os outros módulos (NeuroGestão, NeuroChat, NeuroCompras, etc.) de um único lugar, dependendo de seus níveis de acesso e permissão.

## 🛠️ Tecnologias Utilizadas
*   **HTML5 / CSS / JS Nativo**: Front-end estático contendo o arquivo `index.html` e assets relacionados, sem necessidade de build complexo.

## ⚙️ Como Executar
1. Este projeto pode ser servido por qualquer servidor web estático (Nginx, Apache) ou via Live Server em desenvolvimento.
2. Basta abrir o arquivo `index.html` no navegador.

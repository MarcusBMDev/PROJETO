# 📣 SolicitaMKT

**SolicitaMKT** é o microserviço de abertura e acompanhamento de tickets (chamados) direcionados ao setor de **Marketing e Comunicação**.

Permite que as unidades solicitem criação de artes, postagens, campanhas, edição de vídeos, uniformes ou brindes. O sistema conta com workflow de aprovação, atribuição de designers e entrega de materiais.

## 🛠️ Tecnologias Utilizadas
*   **Node.js**: Stack principal baseado no `package.json`.

## ⚙️ Como Executar
1. Entre na pasta `SOLICITAMKT`.
2. Instale os módulos: `npm install`
3. Execute o serviço: `npm start`
